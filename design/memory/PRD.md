# DLIMS Punjab Homepage Clone - PRD

## Overview
A pixel-perfect visual clone of the DLIMS Punjab homepage (https://dlims.punjab.gov.pk/) built as a static frontend-only website using React with Tailwind CSS.

## Sections Implemented
1. **Header** - Top green bar + Navigation with DLIMS/Dastak logos
2. **Hero Section** - Dastak branding, umbrella illustration, floating license type cards (Urdu)
3. **Features Section** - 3 feature cards (Effortless Application, Real-Time Tracking, Secure Payments)
4. **About Section** - DLIMS description with illustration
5. **Download Section** - App promotion with phone mockup and QR codes
6. **Process Section** - Tabbed interface (Learner/Regular/International) with 5-6 step process
7. **FAQ Section** - 6 accordion-style FAQ items
8. **Footer** - Green curved background, Dastak logo, contact information

## Design Specs
- Primary green: #3FAE2A
- Dark green: #2E7D32
- Gold accent: #c8a84e
- Font: Poppins (Google Fonts)
- Responsive: Desktop + Mobile

## Status
- Frontend-only, non-functional UI clone
- All data is MOCKED in `/src/data/mockData.js`
- No backend integration needed
